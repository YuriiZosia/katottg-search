# Пошук КАТОТТГ | YCsys 🇺🇦

Швидкий, легкий та сучасний веб-інструмент для миттєвого пошуку повної інформації про населені пункти України згідно з офіційним Кодифікатором адміністративно-територіальних одиниць та територій територіальних громад (**КАТОТТГ**).

Утиліта повністю працює на стороні клієнта (Client-side HTML/JS), забезпечуючи максимальну швидкість роботи без затримок на запити до серверів чи баз даних.

🔗 **Живий інструмент доступний за посиланням:** [Перейти до пошуку КАТОТТГ](https://yuriizosia.github.io/katottg-search/)

---

## 🎯 Основна функція

Вам достатньо ввести назву будь-якого міста, селища чи села (наприклад, `Вільне`), і система миттєво згенерує динамічний список усіх знайдених збігів із повною ієрархічною структурою:
`Область, Район, Територіальна громада + її тип, Скорочений тип н.п. + Назва`

**Приклад результату:**
> `Дніпропетровська область, Новомосковський район, Вільненська сільська територіальна громада, с. Вільне`

---

## ⚡️ Ключові можливості

- 🚀 **Миттєвий пошук ($O(1)$ / швидка фільтрація):** Весь масив даних (майже 30 000 населених пунктів) зашитий у легкий структурований JavaScript-масив, що дозволяє отримати результати за 1-2 мілісекунди.
- 📱 **Сучасний адаптивний інтерфейс:** Чистий мінімалістичний дизайн із великим полем пошуку та підтримкою будь-яких пристроїв (десктоп, планшет, смартфон).
- 📋 **Копіювання в один клік:** Наявна кнопка для миттєвого копіювання результатів пошуку у вигляді **форматованого масиву рядків (JSON)** для подальшого використання у коді чи документах.
- 📂 **Автономність:** Працює без баз даних (SQL/PostgreSQL) та сторонніх API. Один HTML файл + один JS файл з даними.

---

## 🛠 Технологічний стек

- **Frontend:** Pure HTML5, CSS3 (Сучасні CSS змінні, Flexbox), Vanilla JavaScript (ES6+, методи `filter()`, `JSON.stringify()`).
- **Data-prep:** Python-скрипт (для початкової конвертації офіційних ієрархічних даних КАТОТТГ з формату CSV/Excel у плаский JS-масив).

## ⚙️ Як розгорнути локально

1. Склонуйте репозиторій або просто завантажте файли проекту.
2. Переконайтеся, що файл даних `data.js` знаходиться в одній папці з `index.html`.
3. Просто відкрийте `index.html` у будь-якому сучасному браузері. Ніяких серверів (Node.js, Apache тощо) запускати не потрібно.

## 📝 Структура даних (`data.js`)

Дані зберігаються у вигляді компактного масиву масивів для зменшення загальної ваги файлу:
```javascript
const katottgData = [
    ["Дніпропетровська", "Новомосковський", "Вільненська", "сільська", "с.", "Вільне"],
    ["Київська", "Фастівський", "Томашівська", "сільська", "с.", "Вільне"]
];

```
